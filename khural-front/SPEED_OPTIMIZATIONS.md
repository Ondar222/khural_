# Оптимизация производительности

## Примененные оптимизации

### 1. Vite конфигурация
- Агрессивная минификация Terser (3 прохода)
- Code Splitting на vendor чанки
- target: 'esnext' для современных браузеров
- CSS code splitting
- Удаление console.* вызовов

### 2. HTML оптимизации
- Предзагрузка критических ресурсов (preconnect, dns-prefetch)
- Асинхронная загрузка шрифтов
- Critical CSS для спиннера
- Оптимизированный loading spinner

### 3. Удаление лишнего
- Убран http-vendor чанк
- Убран defer у script

## Рекомендации для бекенда

1. Включите Gzip/Brotli сжатие
2. Настройте кэширование статики (1 год)
3. Используйте CDN (Cloudflare)
4. Включите HTTP/2

## Ожидаемый результат

- Первый контент: 1-2 сек (было 2-3 сек)
- Полная загрузка: 3-4 сек (было 5-8 сек)
- Размер JS: ~1.2 MB (было ~1.5 MB)
